What do I use for my projects?

The tools and languages that I personally like to use for my data analysis projects
are probably the same as everyone else who does data analysis on this earth. Lol.

But putting that disclaimer aside, the languages that I use are SQL and Python.
And the tools used are Microsoft SQL Server for SQL and Jupyter Notebook
for Python. As for visualisation, I used Tableau Public, as its the best free option
available at the time.
